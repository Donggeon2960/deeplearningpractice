# 인공지능 기초 Term Project 중간보고

## 1. 주제
딥러닝(CNN) 기술을 활용한 흉부 X-ray 폐렴 자동 분류 모델 개발

## 2. 선정 배경

**사회적 중요성**: 폐렴은 전 세계적으로 5세 미만 아동 사망 원인의 약 15%를 차지하는 주요 질병으로, 조기 발견과 치료가 환자의 예후에 결정적인 영향을 미칩니다.

**현실적 문제점**: 흉부 X-ray는 가장 보편적인 진단 도구이지만, 영상 의학과 전문의의 숙련도나 피로도에 따라 미세한 병변을 놓치는 휴먼 에러(Human Error)가 발생할 수 있습니다. 특히 팬데믹과 같은 의료 자원 부족 상황에서는 신속한 1차 선별(Screening) 도구가 절실합니다.

**기술적 적합성**: 이번 학기 '인공지능 기초' 수업에서 학습한 CNN은 이미지 내의 패턴과 특징을 추출하는 데 탁월한 성능을 보입니다. 기본 CNN 모델 또는 VGG16과 같은 사전 학습된 모델의 전이학습(Transfer Learning)을 활용하여, 딥러닝이 실제로 의료 영상 판독을 얼마나 효과적으로 보조할 수 있는지 확인해보고자 이 주제를 선정하였습니다.

## 3. 예상 결과

### [구현 계획]

**데이터셋 활용**: Kaggle의 검증된 오픈 데이터셋인 'Chest X-Ray Images (Pneumonia)'를 사용하여 총 5,000장 이상의 정상(Normal) 및 폐렴(Pneumonia) X-ray 이미지를 학습 데이터로 구축합니다.

**전처리 과정**: 
- 이미지 크기 규격화(Resizing)
- 픽셀 값 정규화(Normalization)
- 데이터 불균형 해결을 위한 데이터 증강(Data Augmentation) 적용

**모델 구조**: 컨볼루션 층(Convolution Layer)과 풀링 층(Pooling Layer)을 거쳐 특징을 추출한 뒤, 완전 연결 층(Fully Connected Layer)을 통과하여 최종적으로 시그모이드(Sigmoid) 함수를 통해 '정상'과 '폐렴' 두 가지 클래스로 분류하는 모델을 설계합니다.

### [성능 목표]

**정량적 평가**:
- 테스트 데이터에 대해 정확도(Accuracy) 90% 이상 달성
- 의료 진단의 특성상 False Negative를 최소화하기 위해 Precision, Recall, F1-score를 함께 평가
- 학습 곡선(Learning Curve)을 통해 과적합(Overfitting) 여부 확인

**기대 효과**: 
- 의료 현장에서 전문의의 1차 선별 보조 도구로 활용 가능성 검증
- 다만, 최종 진단은 반드시 전문의의 판단이 필요하며, 본 모델은 보조 도구로서의 역할을 목표로 합니다.
